# Material Slide Toggle

[mat-slide-toggle](https://material.angular.io/components/slide-toggle/overview) is an on/off control that can be toggled via clicking.

```
 <mat-slide-toggle> Slide </mat-slide-toggle>
```

### Helper classes

`.contained` changes the style and sets the toggle thumb inside the slide bar

```
 <mat-slide-toggle class="contained"> Slide </mat-slide-toggle>
```
