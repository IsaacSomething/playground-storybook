# Material button

[Angular Material buttons](https://material.angular.io/components/button/api) are native `<button>` or `<a>` elements enhanced with Material Design styling and ink ripples. <br />

`import { MatButtonModule } from '@angular/material/button';`
`import { MatIconModule } from '@angular/material/icon';`

```
<button mat-button> Default </button>
<button mat-flat-button> Flat </button>
<button mat-raised-button> Raised </button>
<button mat-stroked-button> Stroked </button>
<button mat-icon-button> <mat-icon>save</mat-icon> </button>
```
