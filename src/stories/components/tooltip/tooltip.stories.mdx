# Material Tooltip

[The Angular Material tooltip](https://material.angular.io/components/tooltip/api) provides a text label that is displayed when the user hovers over or longpresses an element..

`import {MatTooltipModule} from '@angular/material/tooltip';`

### API

```
matTooltip
```

```
matTooltipPosition

```
matTooltipClass

```

```
